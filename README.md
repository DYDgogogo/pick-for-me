# 帮我选 - Pick For Me

🎯 选择困难症克星！纠结什么？转一下就知道了！

## 功能特点

- 🎡 精美的转盘动画，帮你做随机选择
- ✏️ 自定义选项，想选什么加什么
- 📋 预设模板，常用场景一键切换
- 📜 历史记录，回顾每次选择
- 💾 本地存储，选项和历史自动保存

## 技术栈

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
